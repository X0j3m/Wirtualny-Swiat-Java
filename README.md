# Wirtualny-Swiat-Java

Symulator napisany w języku Java wraz z warstwą graficzną w JavaFX. Projekt powstał na potrzeby studenckich zajęć projektowych z przedmiotu Programowanie Opbiektowe.

Na planszy znajduje się człowiek którym można się poruszać, oraz inne zwierzeta i rośliny wchodzące ze sobą w interakcję.
